# Modelling the ability to humanity to be multiplanetary sustainable

Light approximate simulation tool to evaluate if there is enough ressources on earth to let humanity reach a technology level 
that allows multpilanetary deployment of the specie.

To reach multiplanetart stage, humanity has to dig into earth resources + renewables.
If it fails because its not possible, humanity burns a piece of rope by both sides quicker and quicker.

To succeed, can be helpful to have an idea of which kind of specie "dynamic" can get it there ... when ... at which cost ?

  LOCAL
